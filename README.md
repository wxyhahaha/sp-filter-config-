## 说明
安装 node 
获取网络配置命令 node getconfig
修改配置命令 node saveconfig
# 描述
为了方便提交json配置文件，简单做了一个小插件
# 安装
npm i sp-fitler-config -g
# 命令
| 命令 | 说明 |
| :--- | :--- |
| `public` | 查看所有命令 |
| `public get` | 获取网络上json |
| `public save` | 保存本地保存json |
